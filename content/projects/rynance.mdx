### Introduction
RyNance is a responsive web-based application designed to improve financial literacy among Singaporean youth. It offers tools like a Retirement Calculator, Budget Planner, Undergraduate Salary Comparison tool, and a wiki page for additional resources.

### Tech Stack
Built using Flask, Chart.js, HTML, CSS, Python, and SQLite BootStrap5.


### Features
- **User Authentication**: Implemented user authentication to ensure data privacy and security.
![Auth image](https://user-images.githubusercontent.com/80184478/161922643-6317a9a8-c511-4533-842f-e77df1bd6a6a.png)
- **Retirement Calculator**: A tool that calculates the amount of savings needed for retirement based on the [Trinity Study](https://en.wikipedia.org/wiki/Trinity_study).
![Retirement Calculator](https://user-images.githubusercontent.com/80184478/161925384-d65db675-9bf6-4788-b78e-583fc2e1a8a3.gif)
- **Budget Planner**: A tool that helps users plan their monthly budget based on their income and expenses using Chart.js.
![Budget Planner](https://user-images.githubusercontent.com/80184478/161924929-c0e418ed-c4c7-405c-b5c1-2ab9ecfed0af.gif)
- **Learn**: A wiki page that provides resources and information on financial literacy.
![Learn](https://user-images.githubusercontent.com/80184478/161925698-9c81da16-f406-44b7-bd08-89f2fb88c6c9.gif)
- ** Compare**: A tool that compares the average salary of different undergraduate courses in Singapore using singstat api.
![Compare](https://user-images.githubusercontent.com/80184478/161926536-29e589e3-6acd-4352-9f9b-4cf170975b6f.gif)
